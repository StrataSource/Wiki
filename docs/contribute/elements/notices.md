# Notices

Notices can be used to call out important or useful info.

## Default Notice

```markdown
> This is the default notice.
```

> This is the default notice.

## Note

```markdown
> [!NOTE]
> This is a note.
```

> [!NOTE]
> This is a note.

## Tip

```markdown
> [!TIP]
> This is a tip.
```

> [!TIP]
> This is a tip.

## Warning

```markdown
> [!WARNING]
> This is a warning.
```

> [!WARNING]
> This is a warning.

## Caution

```markdown
> [!CAUTION]
> This is a warning, but scary.
```

> [!CAUTION]
> This is a warning, but scary.

